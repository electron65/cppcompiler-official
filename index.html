E65A Architecture

E65A is an 8bit CPU currently developing by electron65. This project started when electron65 was trying to modify the Z80 CPU made by Zilog. Although Z80 may not be as sophisticated as to contemporary CPU's, it's still harder to design a CPU from scratch.
